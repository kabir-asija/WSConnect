# Resume Chat App (React + Node + Express + Socket.IO)

A beginner-friendly real-time chat app to showcase WebSocket skills on your resume.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Realtime: Socket.IO

## Project Structure
- `client/` -> React app
- `server/` -> Express + Socket.IO server

## 1) Run the server
```bash
cd server
npm install
npm run dev
```

Server runs on `http://localhost:5000`.

## 2) Run the client
```bash
cd client
npm install
npm run dev
```

Client runs on `http://localhost:5173`.

## Features
- Join chat with your name
- Live messaging with Socket.IO
- Typing indicator
- Online users list
- Auto-scroll to latest message

## Resume Tip
You can mention:
- "Built a real-time chat app using Socket.IO with typing events and active user tracking."
- "Implemented event-based WebSocket communication between React frontend and Express backend."
